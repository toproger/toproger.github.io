Какой язык программирования выбрать в 2020 году

Этот вопрос мучает начинающих и заядлых программистов. В действительности, а какой выбрать?

Ответ зависит от того чем вы планируете заниматься. Это может быть WEB разработка, игры или десктопные программы, а ещё этот выбор зависит от платформы.


В это статье мы разберём популярные и перспективные языки программирования, но не все (т. к. их огромное кол-во)



1) C#

Csharp создан компанией Microsoft в 2001 году. Он отлично подходит к платформе Windows.

Преимущества:


1) Десктопные программы на Windows

2) Можно писать Android программы благодаря Xamarin

3) Хорошая документация

4) Автоматический сбор мусора

5) Отзывчевое и большое сообщество



2) Java

Java был создан компанией Sun MicroSystems 23 мая 1995 г. и выкуплен компанией Oracle. Java программы транслируются в байт-код Java и воспроизводятся (JVM) виртуальной машиной Java. По сути Java интерплитируемый язык программирования. Программа на java запустится на любой платформе у которой установлена JVM.

Преимущества:


1) Java запустится на любой платформе

2) Хорошая документация

3) Автоматический сбор мусора

4) Подходит под андроид разработку

5) Отзывчевое и большое сообщество



3) C++

Язык C++ появился на свет в 1980 году Бьёрн Страуструп придумал дополнения к языку C для себя, но в итоге понял что это будет новый язык программирования, а он его основатель. Несмотря на возраст языка его используют до сих пор. Для начинающего может показаться сложным

Преимущества:


1) Самый быстрый по компиляции и работе программы

2) На нём можно писать на множество платформ

3) Есть возможность управления памятью

4) На нём можно писать игры, десктопные приложения, различные виды программ

5) Отзывчевое и большое сообщество



4) Python

Python начал разрабатываться в конце 1980 г. Гвидо ван Россумом. Python подойдёт для новичков. На Python существует много библиотек (Дополнений), благодоря этому вы не будете "заново делать велосипед"

Преимущества:


1) Самый простой в освоении

2) Большое кол-во готовых решений

3) Хорошая документация

4) На нём можно написать всё: сервер, десктоп программа, игры и т.д.

5) Отзывчевое и большое сообщество

6) Отзывчевое и большое сообщество

7) Тип переменных определяется автоматически

8) Встроенный помощник по установке пакетов (PIP)

9) Можно делать WEB приложения



5) Ruby

Ruby был создан в 1993 году Юкихиро Мацумото. Его в основном используют для Web разработки. У него C-подобный синтаксис.

Преимущества:


1) Можно использовать в Html

2) Присутствует встроенный отладчик

3) Продвинутые способы работы со строками

4) Есть инструменты для увеличения безопасности



6) php

PhP изобрели 1995 г. Раумусом Лердорфом. В 1994 он создал инструмент на C, который вёл учёт посетителей на его странице. Ему не хватало функционала и он создал PHP.

Преимущества:


1) Простой и понятный

2) Высокая скорость работы

3) Можно разработать отдельно от Html, а потом соединить

4) Поддерживается большинством хостингов

5) Большое кол-во библиотек (Дополнений)



7) JavaScript

JavaScript был создан 1995 г. Бренданом Эйхом. Он внедрил что-то похожее на Scheme в JavaScript. JavaScript в основном используют для дополнения Html страницы функционалом, но его можно использовать как интерплитатор.

Преимущества:


1) Изменяет Html страницы

2) Может работать с Cookie-файлами

3) Может вносить изменения в свой код

4) Добавлять/удалять теги Html документа

5) Изменять стиль Html документа



8) HTML

Html был изобретён в 1986-1991г. Тимом Бернерсом-Ли. Изначально его использовали для обмена научной информацией, но его теперь использует каждый для создания сайта.

Преимущества:


1) Подходит для создания корпуса веб страницы
